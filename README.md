# FixGo – Web-Based Service Management Platform

## Project Overview
**FixGo** is a web-based service management platform developed as a final year academic project.  
It connects **customers**, **service providers**, **companies**, and **administrators** in one secure and user-friendly system.

The platform helps users easily **find**, **book**, **pay**, and **review** service providers while ensuring trust through verification and transparent feedback mechanisms.

## Problem Statement
In Sri Lanka, finding reliable and verified service providers is difficult.  
Existing solutions are:
- Not user-friendly  
- Lack trust and verification  
- Do not support quotation comparison or emergency services  

FixGo addresses these issues by offering a **centralized, secure, and intelligent platform**.

---

## Key Features
- User registration and role-based login
- Service search and filtering by location, price, and rating
- Real-time quotation requests and booking system
- Emergency service request option
- AI chatbot for 24/7 assistance
- Secure online payment gateway
- Ratings, reviews, and complaint handling
- Admin dashboard for system monitoring
- Company portal for business service management

---

## User Roles
- **Customer:** Search services, book providers, make payments, give reviews
- **Service Provider:** Manage profile, accept bookings, send quotations
- **Company:** Post jobs and manage multiple service requests
- **Admin:** Verify users, manage complaints, monitor system activity
